🚀 Real-Time Data Pipeline & Dashboard

📌 Project Overview

This project demonstrates an end-to-end real-time data pipeline that ingests, processes, stores, and visualizes streaming data. It simulates live transaction data and provides real-time insights through an interactive dashboard.


---

🧠 Key Features

Real-time data streaming using Kafka

Stream processing using Spark

Data storage in MySQL/MongoDB

Interactive dashboard using Streamlit/Power BI

Anomaly detection for unusual transactions

Scalable and modular architecture



---

🏗️ Architecture

Data Source → Kafka → Spark Streaming → Database → Dashboard


---

⚙️ Tech Stack

Python

Apache Kafka

Apache Spark

MySQL / MongoDB

Streamlit / Power BI



---

📂 Project Structure

real-time-pipeline/
│
├── producer/              # Generates real-time data
├── consumer/              # Consumes Kafka data
├── spark_processing/      # Stream processing logic
├── database/              # Database connection & storage
├── dashboard/             # Visualization layer
├── models/                # ML models (optional)
└── README.md


---

🔄 Workflow

1. Producer generates real-time data


2. Kafka streams the data


3. Spark processes and transforms data


4. Data is stored in database


5. Dashboard visualizes insights in real-time




---

📊 Use Cases

Financial transaction monitoring

---

🚀 How to Run

1. Start Kafka server


2. Run producer script


3. Start Spark streaming job


4. Run dashboard application
